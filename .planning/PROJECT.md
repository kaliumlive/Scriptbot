# Scriptbot

## Mission
To provide a frictionless, agentic engine for short-form content creation and distribution, starting with Instagram. Scriptbot aims to automate the entire pipeline from script generation to verified social media publishing.

## Vision
A world where high-quality content creation is a 1-click agentic flow, seamlessly handling scriptwriting, video generation, and platform-native publishing while maintaining authentic creator resonance.

## Core Value Proposition
- **Agentic Automation**: Intelligent scripting and video processing pipelines.
- **Frictionless Publishing**: One-click connection and distribution to Meta, LinkedIn, and YouTube.
- **Creator-Centric**: Designed for rapid iteration and high-impact social media presence.

## Technical Identity
- **Stack**: Next.js 16 (App Router), Supabase, AI SDK (Gemini/Groq), FFmpeg/Puppeteer (Video Processing).
- **Architecture**: Server-centric agentic pipelines with robust OAuth-based platform integrations.
- **Deployment**: Vercel (Global Edge Network) with Supabase (ap-south-1).
